Abstract

EIDORA is an open-source desktop application for visual exploration of vector representations. It provides a local workflow environment for embedding space analysis: importing corpora, computing vector embeddings, running projection and clustering algorithms, creating interactive visualizations with custom filters and annotations, and producing figures for publication. The software is designed for researchers working with collection data across different modalities, including image, text, audio, video, 3D, and multimodal combinations. In cultural collections, such media are often accompanied by contextual information such as time, authorship, text labels, or other rich metadata. EIDORA enables scholars to draw from their own domain expertise and judgment by providing an intuitive project environment for investigating and interpreting structures, patterns, trends, shifts, and anomalies in vector representations, turning exploratory analysis into reproducible and reusable research workflows.
